# 목표

## 진짜 만들고 싶은 것 (v2.0)

** 그리면서 쓰고, 쓰면서 그리고, 사진도 붙이는 앱.**

종이 노트에 그림그리고 글씨쓰고 하는 일을 그대로 디지털로 옮겨 보자.

### 타깃 사용자

- 시각적으로 생각하는 사람
- 메모할 때 글만으로는 부족하다고 느끼는 사람

### 핵심 차별화 (검증 중)

- 가볍고 빠른 네이티브 앱
- 자유 캔버스와 선형 흐름의 균형

### 결과물 포맷

마크다운 파일이 진실의 원천.
드로잉은 SVG 또는 PNG로 저장, 마크다운으로 참조.

## 프로토타입 (v0.5 Alpha)

비주얼 노트로 가기 위해 통과 해야 할 단계

### 먼저 할 것

1. v1.0을 출시할 수 있다는 걸 자기 자신에게 증명
2. 비주얼 노트의 핵심 컴포넌트인 그리기 엔진을 미리 검증
3. Avalonia, .NET 사용, 다른 언어가 고민되지만 선택과 집중
4. 실제 사용자 피드백을 통해 v2.0의 방향 검증

### 중요

- v0.5 출시 전에 v2.0 기능 슬쩍 끼워 넣지 않기
- Obsidian, Notion, OneNote와 비교하면서 위축되지 않기
- v0.5 알파의 상업적 가능성 걱정하지 않기

## 성공이 무엇인가

### v0.5 알파 (6개월 후)

- 친구 5명이 실제로 그림 그리는 데 사용
- 한 명이라도 "이거 글도 쓸 수 있으면 좋겠다"라고 말함 (v2.0 신호)
- 코드 베이스가 v2.0 진입 준비된 상태

### v1.0 Windows 출시 (9개월 후)

- Microsoft Store 출시 완료
- v2.0 시작할 동력이 살아있음

### v2.0 비주얼 노트 (24~36개월 후)

- 본인이 매일 쓰는 도구가 됨
- "이거 없으면 안 됨" 사용자 50명 이상

## 기술 스택과 결정 사항

- **언어**: C# (.NET 10)
- **UI**: Avalonia UI 12
- **그래픽**: SkiaSharp
- **라이선스**: PolyForm Noncommercial 1.0.0
- **출시 순서**: Windows → Linux → Mac → Android → iPad
